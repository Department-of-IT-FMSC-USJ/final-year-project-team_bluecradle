wifi_off

Offline-First Clinic Sessions

PHMs in areas like Deraniyagala experience complete network blackouts. BlueCradle's IndexedDB architecture pre-syncs all infant profiles to the device before the clinic session. Every measurement, vaccination, and alert is written locally — nothing waits for a network call.

  • check_circle Auto-detects latency >3 seconds and switches to offline mode.
  • check_circle Background Sync API fires automatically on Wi-Fi restoration — no PHM action needed.
  • check_circle Priority queue: SAM/MAM alerts transmitted first, routine admin last.

Real-Time WHO Growth Monitoring

Immunization Tracking & Defaulter Flagging

How parents access BlueCradle

Parents register independently using the Personal Health Number (PHN) printed on their physical CHDR booklet. No PHM involvement required. One shared household credential gives both parents read-only access to their child's digital record.
